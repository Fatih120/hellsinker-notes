HS1.009 item technical documentation

Array start address at 7BA6E8 (pointer at 78BE8C)
Update routines locations are stored at CA9514
Routines to generate items:
- 511F8C:
	(TBA)
- 51201C:
	Pops 2 values.
	ECX=Y coordinate
	EDX=X coordinate
	EAX=type of item

Each item holds 0x8C/140 bytes of data.
The item array holds up to 512 items.

[item+04]: collection radius
[item+0C]: timer
[item+10]: type
[item+14]: X position
[item+18]: Y position
[item+40]: Luna crystal maturity level (TOKEN multiplier for Luna crystals)

Types:
0x00	SOL piece
0x01	LUNA crystal
0x02	STELLA UP item
0x03	STELLA DOWN item
0x04	Ingot
0x05	Extend item
0x06	LUNA chaff
0x07	TERRA token
0x08	Shrine of Farewell blue crystal
0x09	Shrine of Farewell red crystal
0x0A	Bullet cancel LUNA crystal
0x0B	Comet relic (spherical item generated upon death)
0x0C	ReVIVE item
0x0D	LUNA droplet (generated when destroying an enemy with bullets over it)
